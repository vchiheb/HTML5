How to Add a Code Snippet to the Visual Studio Code Editor

Notes

Snippets can be global or local. Here we will see how to create a global snippet.

How To

  1. Click the gear icon in the bottom left corner of the app.
  2. Select "Snippets".
  3. Type in the name of the global file in the input field at the top of the window and hit enter to enter your new global snippets file.
  4. Follow the instructions in the comments. Commenting out the sample snippet definition and make it your own. You can make as many snippet defintions as required. Just rename the prefix and make sure that there is only one set of curly braces around all of the snippets.
