Visual Studio Code Extensions

Notes

This page include a quick start how to for getting started with Visual Studio Code extensions and a lists visual studio code extensions that I like to use.

How To

  1. From the left hand navigation bar click the boxes kind of icon at the bottom of this navigation bar. From here you can see what extensions you have installed.
  2. Enter the name of the extension in the input box at the top of the extensions pane.
  3. From the main pane click the install button.
